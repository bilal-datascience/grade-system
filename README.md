# 📊 Grade System — Python

A simple Python program that calculates student grades based on marks.

## Features
- Takes marks as input (0-100)
- Calculates grade automatically
- Handles invalid input
- Simple and clean code

## 📋 Grading Criteria
| Marks  | Grade |
|--------|-------|
| 90-100 | A+    |
| 80-89  | A     |
| 70-79  | B     |
| 60-69  | C     |
| 50-59  | D     |
| 0-49   | Fail  |
